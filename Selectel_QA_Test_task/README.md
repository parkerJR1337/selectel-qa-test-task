# Selectel QA Test Task

Тестовое задание на позицию QA Intern.

Проект состоит из двух частей:
1. Исследовательское тестирование
2. Автоматизированные тесты


## Часть 1 — Исследовательское тестирование

Результаты тестирования процесса регистрации оформлены в отдельном документе.

Расположение:

Task1_exploratory/

Документ содержит:
- список проверок
- ожидаемые и фактические результаты
- статус выполнения
- UX-наблюдения и рекомендации по улучшению процесса регистрации


## Часть 2 — Автоматизированные тесты

Автотесты главной страницы сайта:  
https://selectel.ru

Технологии:
- Python 3
- Selenium WebDriver
- Pytest
- ChromeDriver

Расположение:

Task2_Autotests/

Структура автотестов:

Task2_Autotests/
│
├── tests/
│     └── test_main_page.py
│
├── requirements.txt
└── README.md

---

## Реализованные проверки

1. Проверка открытия главной страницы
2. Проверка наличия кнопки «Войти»
3. Проверка наличия header (верхнего меню)
4. Проверка возможности прокрутки страницы до конца
5. Проверка, что заголовок страницы (title) не пустой



## Установка зависимостей

Перейти в папку 2_Autotests и выполнить:

pip install -r requirements.txt

---

## Запуск тестов

В папке 2_Autotests выполнить:

pytest -v

---

## Примечание

Тесты реализованы с использованием явных ожиданий (WebDriverWait) для корректной работы с динамически загружаемыми элементами страницы.
