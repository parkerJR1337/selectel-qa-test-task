# Autotests for selectel.ru

## Реализованные проверки
1. Открытие главной страницы
2. Проверка наличия кнопки "Войти"
3. Проверка наличия header (верхнего меню)
4. Проверка возможности прокрутки страницы до конца
5. Проверка, что заголовок страницы (title) не пустой

## Используемые инструменты
- Python
- Selenium
- Pytest
- ChromeDriver
## Как запустить

1. Установить Python
2. Выполнить: pip install -r requirements.txt
3. Выполнить: pytest -v
